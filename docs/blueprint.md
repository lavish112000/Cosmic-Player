# **App Name**: Cosmic Player

## Core Features:

- Animated UI/UX: Dynamic and sleek user interface with a modern animated design and UX.
- Interactive Buttons: Floating buttons with a bouncing animation on click for enhanced interactivity.
- Cosmic Background: Aesthetic 4D cosmic space and motion theme for the video player background.
- Rolling Navigation Bar: Floating LC icon on the top left which rolls out a navigation bar on hover/click.
- Playback Controls: Comprehensive settings for audio and video playback including track selection, device selection, subtitle options, zoom, aspect ratio, crop, and speed control.
- File Management: Versatile file management with options to open files, folders, and recent media.
- Full Screen Playback: Built-in full screen mode that hides UI elements when no movement is detected. Also incorporates codec for playing high quality files smoothly

## Style Guidelines:

- Primary color: Deep space purple (#4A148C) to reflect the cosmic theme.
- Background color: Desaturated dark purple (#1E122A) to ensure the interface elements stand out against the cosmic backdrop.
- Accent color: Electric indigo (#663399) for interactive elements and highlights.
- Font: 'Space Grotesk' sans-serif for headlines, and 'Inter' sans-serif for body, for a modern, techy feel.
- Use glowing, neon-style icons related to video controls and settings, consistent with the cosmic theme.
- Floating and semi-transparent UI elements layered over the cosmic background.
- Smooth, animated transitions and bouncing effects for button interactions to enhance user engagement.
